- 📊 تقرير أسبوعي تفصيلي لكل عادة
- 🔥 سلسلة الأيام المكتملة (Streak)

## طريقة التشغيل

### الخطوة 1 — إنشاء البوت على تيليغرام
1. افتح تيليغرام وابحث عن **@BotFather**
2. أرسل `/newbot`
3. اختر اسم للبوت (مثال: My Habits Bot)
4. اختر username ينتهي بـ `bot` (مثال: `my_habits_bot`)
5. **احفظ التوكن** الذي سيعطيك إياه

### الخطوة 2 — رفع البوت على Railway (مجاني)

#### أ) إنشاء حساب
- اذهب إلى [railway.app](https://railway.app)
- سجّل دخول بحساب GitHub

#### ب) رفع الكود
1. ارفع هذا المجلد على GitHub repo جديد
2. في Railway: **New Project → Deploy from GitHub repo**
3. اختر الـ repo

#### ج) إضافة المتغيرات
في Railway → Variables → أضف:
```
BOT_TOKEN = توكن_البوت_من_BotFather
```

#### د) إضافة ملف Procfile
أنشئ ملف اسمه `Procfile` في نفس المجلد:
```
worker: python bot.py
```

### الخطوة 3 — تشغيل البوت
- بعد الرفع، سيشتغل البوت تلقائياً
- افتح تيليغرام وابحث عن username البوت
- أرسل `/start`

## التشغيل المحلي (للتجربة)

```bash
# تثبيت المكتبات
pip install -r requirements.txt

# وضع التوكن مباشرة في bot.py أو عبر متغير البيئة
export BOT_TOKEN="توكن_البوت_هنا"

# تشغيل
python bot.py
```
